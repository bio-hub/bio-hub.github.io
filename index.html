Goals

DaVinci Application performs several text mining analysis starting from scientific literature, in order to help scientist and health professionals to translate all information into a clear perspective.

What we do

Services and analyses

Syntax Analysis of Scientific Literature

Gene Panel

#3

Drug Panel

#4

Tissue Panel

Host Tissue and cell information

4. Transmission
